# Pokemon NFT Gallery - 快速部署指南 V3

## 问题说明

**问题**: 运行部署脚本后，Vercel登录窗口没有弹出，导致无法完成登录。

**原因分析**:
1. Vercel CLI在某些情况下不会自动打开浏览器
2. 需要手动复制登录链接到浏览器
3. 原脚本窗口可能一闪而过，没有显示完整的提示

### 方法 2: 分步执行（推荐新手）

- 查看当前环境是否准备就绪

- 双击运行：`login-helper-v2.bat`

  - 手动复制该链接到浏览器打开
  - 登录或注册Vercel账户（完全免费）
  - 验证邮箱后，回到终端按任意键继续

**第3步：部署项目**
- 登录成功后，双击运行：`deploy-2-v2.bat`
- 按照提示完成部署
- 部署完成后会显示你的网站URL

### 方法 3: 手动执行（推荐高级用户）

如果自动脚本遇到问题，可以手动执行：

**1. 检查 Vercel CLI**
```batch
vercel --version
```

**2. 登录 Vercel**
```batch
vercel login
```
- 终端会显示登录链接
- 复制链接到浏览器
- 完成登录流程

**3. 安装依赖**
```batch
npm install
```

**4. 构建项目**
```batch
npm run build
```

**5. 部署到 Vercel**
```batch
vercel --prod
```

## 常见问题

### Q1: 登录链接没有自动打开浏览器怎么办？

**A**: 这是正常现象。请手动复制登录链接到浏览器打开。

### Q2: 登录链接在哪里显示？

**A**: 运行 `login-helper-v2.bat` 或执行 `vercel login` 后，终端会显示类似这样的链接：

```
> Log in with [https://vercel.com/cli/login]
```

复制方括号中的链接到浏览器即可。

### Q3: Vercel账户需要付费吗？

**A**: 不需要。Vercel有免费计划，足够个人使用。

### Q4: 部署失败怎么办？

**A**:
1. 运行 `check-status-v2.bat` 检查环境
2. 查看错误信息
3. 尝试手动执行命令
4. 检查网络连接

### Q5: 如何查看部署的网站？

**A**: 部署完成后，终端会显示你的网站URL，格式通常是：

```
https://pokemon-nft-gallery-xxx.vercel.app
```
